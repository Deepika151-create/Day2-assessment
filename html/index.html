Full Stack Web Developement-A Passionate Course

************************************************************

Full Stack developement refers to the development of both front end(client side) and back end(server side) portions of web application. Full stack web developers have the ability to design complete web applications and websites. They work on the frontend, backend, database and debugging of web applications or websites.

Frontend Technologies

  1. HTML(Hyper Text Markup Language)
  2. CSS(Cascading Style Sheet)
  3. JS(Java Script)

Backend Languages

  • Angular JS
  • React JS
  • Python and more...
